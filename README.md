> Built custom blockchain scanners, indexers, and developer tools for the **MSC Layer 2 EVM chain** — enabling real-time transaction tracking, block exploration, and on-chain data analytics.

- ⚙️ Custom RPC integration & block event listeners
- 📊 Real-time indexing pipeline with PostgreSQL
- 🔍 Transaction, wallet, and contract monitoring dashboard

---

### 💳 FaradPay — Crypto Payment Gateway
> Engineered a **multi-currency crypto payment gateway** with secure webhook and callback systems for merchants and exchanges.

- 🔐 HMAC-signed callback verification system
- 💱 Multi-coin support: ETH, BNB, USDT, custom tokens
- 📡 Real-time payment confirmation via on-chain listeners
- 🏦 CEX-ready wallet management & balance tracking

---

### 📈 DEX & CEX Exchange Platforms

- 🔄 AMM-based DEX with Solidity smart contracts
- 📋 CEX with limit/market order books & matching engine
- 👤 KYC-ready user management & admin control panels
- 🔗 Full Web3 wallet integration (MetaMask, WalletConnect)
